Prework Study Guide

HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The footer content usually contains navigation, dates, authors of the site
  • Elements and tags are used interchangeably as terminology, however, 'elements' are everything that exists within the 'tags'
  • Headings create hierarchy for tehe reader and are labelled 1 for largest all the way to 6 for smallest

CSS

  • CSS stands for Cascading Style Sheets
  • CSS gives your HTML colour, fonts, and stylistic qualities
  • The HTML code is pointed towards your CSS within the head of the index

Git

  • Git is a repository for your files
  • Your Git is where you upload yoru work and download your work from the cloud
  • You can create branches from your MAIN Git before merging changes for undestructive editing
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript allows for interactivity on your website
